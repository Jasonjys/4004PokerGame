# Instructions
### To Compile and Run
1. `cd 4004PokerGame`
2. `mvn package` to build the project (make sure you have maven installed)
3. `java -cp target/4004PokerGame-1.0-SNAPSHOT.jar com.comp4004.PokerGame < input.txt`

### To Run All Tests
1. `mvn test`
